\addcontentsline{toc}{chapter}{Resumen}

\begin{abstractsESP} 

Esta tesis propone una infraestructura genérica para el desarrollo
de aplicaciones distribuidas y paralelas sobre la Web. Esta
infraestructura está orientada a permitir que todos los usuarios de
Internet sean capaces de participar en la ejecución de aplicaciones
distribuidas, usando una configuración simple pero con claras
garantías de seguridad.  Nuestra propuesta se basa en el uso de
protocolos disponibles en la {\it World Wide Web} y {\it applets} Java
exclusivamente.  Por ende, el usuario que desee participar de esta
infraestructura s\'{o}lo requerirá de un navegador convencional que
tenga habilitado su JRE ({\it Java Runtime Environment})
correspondiente.  Nuestra propuesta difiere de las anteriores por su
simplicidad y mejor desempeño.  Adicionalmente, nuestro trabajo
incluye una evaluación detallada de la ejecución de aplicaciones
reales sobre nuestra infraestructura.

\end{abstractsESP}
